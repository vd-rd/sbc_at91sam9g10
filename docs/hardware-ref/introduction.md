# Introduction

The Glasnost M9G10 is an open hardware IoT platform based on the Microchip AT91SAM9G10 SoC. It is designed for projects that require connectivity, reliability, and full control over both software and hardware. The board runs Linux and is intended for embedded applications where flexibility and openness are essential.

## Key Features

- ARM926EJ-S processor @ 266 MHz
- Up to 128 MB low-power SDRAM
- NOR flash for boot and storage
- USB host/device support
- Multiple UARTs, SPI, I²C, GPIO
- Ethernet and other connectivity options
- Fully open hardware design with replicable manufacturing files
